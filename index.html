About Kool OS

Kool OS is a Linux distribution designed for both public users and developers. Based on KDE neon, with an Arch variant, it features a custom app store, modern browser stack, and mobile integration.

  • Developer Friendly
  • Mobile Ready (Kool Mobel)
  • Powerful Terminal Tools
  • Global Community

Info

Kool OS supports DEB and Pacman (C-based) package systems. Designed for desktops, mobile, and Steam Deck.

sudo apt update && sudo apt upgrade -y
sudo apt install kde-standard firefox chromium-browser
      
# For Arch-based Kool OS
sudo pacman -Syu
sudo pacman -S plasma firefox chromium
